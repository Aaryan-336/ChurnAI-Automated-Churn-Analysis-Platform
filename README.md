## 🛠️ The Vision

**ChurnAI** is a high-performance, enterprise-grade analytics platform designed to transform raw user data into actionable retention strategies. By leveraging automated machine learning and a dynamic intelligence engine, ChurnAI identifies at-risk accounts before they leave, allowing teams to intervene with precision.

## ✨ Features

- **🚀 Server-First Architecture**: Built with Next.js 14 Server Components for near-instant rendering and a specialized "Shimmering" UI for elite performance.
- **🧬 Dynamic Intelligence Engine**: A proprietary keyword-scanning system that analyzes your dataset's schema in real-time to suggest the most relevant metrics (LTV, Retention Velocity, ARPU).
- **📊 Interactive Analytics**: High-fidelity visualizations powered by Recharts, featuring smooth Framer Motion transitions and automated signal clustering.
- **⚡ Strategic ML Recommender**: Select optimized metrics and run "Strategic Analyses" to simulate the impact of retention interventions on your bottom line.
- **🎨 Premium Visual Experience**: A cohesive dark-mode aesthetic with glassmorphism, accent glows, and responsive layouts.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aaryan-336/ChurnAI-Automated-Churn-Analysis-Platform.git
   cd ChurnAI-Automated-Churn-Analysis-Platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env` file in the root:
   ```env
   DATABASE_URL="file:./dev.db"
   JWT_SECRET="your-secret-key-here"
   ```

4. **Initialize Database**
   ```bash
   npx prisma generate
   npx prisma db push
   npm run seed:demo
   ```

5. **Launch**
   ```bash
   npm run dev
   ```
